Definition

Table of results.

Table options:

Name Value Description
width 100% Chart width
height 500 Chart height
Title PIB en fonction de la population et les pays Chart title
hAxis PIB Chart hAxis
vAxis Population Chart vAxis
bubble fontSize: 11 Bubble size
